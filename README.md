## 📌 프로젝트 개요

<p><b>Find The Answer!</b>는 처음에는 평범한 상식 퀴즈 사이트처럼 진행되다가 갑작스럽게 공포 사이트로 전환되는 웹 기반 퀴즈 게임입니다.</p>
<p>사용자는 퀴즈를 풀며 점점 분위기가 변하는 사이트를 경험하고, 최종적으로 자신의 선택과 플레이 기록을 확인할 수 있습니다.</p>
